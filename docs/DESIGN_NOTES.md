# Design Notes

## 设计定位

本仓库发布的是 PCB-02 光伏 MPPT 可调充电功率板的硬件资料。该板面向光伏输入、电池充电和直流母线供电场景，配合 STM32G474 + RT-Thread 固件实现采样、MPPT/CC-CV 控制、故障处理和上级 CAN 通信。

## 发布内容边界

当前发布内容包括：

- 原理图 PDF。
- Gerber/Drill 生产文件。
- Gerber 解压文件。
- 接口说明、硬件概览、发布检查清单。

当前暂未包含：

- BOM。
- 贴片坐标/CPL。
- 装配图。
- 完整功率测试报告。

## 设计说明

该板的核心设计目标是形成一个独立的 MPPT 功率控制节点：

```text
光伏输入/外部 DC 输入 -> MPPT DC-DC 功率板 -> 电池/BMS/直流母线
```

板上 MCU 负责本地实时控制和保护，上级系统通过 CAN 下发启停、目标电压、限流、电池类型等参数。功率板本地必须具备安全默认态，不能依赖上级主控才能关断。
